## ✨ Funcionalidades

| Feature | Descrição |
|---------|-----------|
| 🖱️ Cursor personalizado | Ponto + anel que seguem o mouse com animação suave |
| ✨ Partículas animadas | Canvas API com 80 partículas e linhas de conexão |
| ⌨️ Efeito typewriter | Frases que digitam e apagam automaticamente no hero |
| 🎨 Thumbnails animados | Cada card de projeto tem um canvas temático único |
| 👁️ Scroll reveal | Seções aparecem com fadeUp ao entrar na tela |
| 🔢 Count-up | Números nos stats sobem animados ao aparecer na tela |
| 📱 Menu mobile | Drawer lateral com animação e overlay blur |
| 📐 Responsivo | Layout adaptado para desktop, tablet e mobile |
| 🌈 Avatar animado | Anel do perfil com gradiente rotativo |
| ⏱️ Timeline | Jornada de aprendizado com linha do tempo animada |
