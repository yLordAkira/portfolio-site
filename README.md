# 💻 Portfólio — Bruno Lopes

Portfólio pessoal desenvolvido com HTML, CSS e JavaScript puro — sem frameworks, sem dependências.

🔗 **[Ver ao vivo](https://ylordakira.github.io/portfolio)** *(após publicar no GitHub Pages)*

---

## ✨ Features

- Cursor personalizado animado
- Background de partículas com Canvas API
- Efeito typewriter no hero
- Thumbnails animados nos cards de projeto (Canvas)
- Scroll reveal com IntersectionObserver
- Count-up nos stats
- Menu mobile com drawer lateral
- Layout totalmente responsivo

## 🚀 Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/yLordAkira/portfolio.git

# Entre na pasta
cd portfolio

# Abra no navegador (ou use Live Server no VS Code)
open index.html
```

---

## 🌐 Deploy no GitHub Pages

1. Vá em **Settings** → **Pages**
2. Em *Source*, selecione **Deploy from a branch**
3. Escolha a branch `main` e pasta `/ (root)`
4. Salve — em alguns minutos o site estará no ar em:
   `https://ylordakira.github.io/portfolio`
